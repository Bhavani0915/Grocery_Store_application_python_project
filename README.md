# Grocery_Store_application_python_project
Grocery Store Management Application developed with Python, Flask, and MySQL. Designed to streamline grocery store operations by managing products, inventory, and store records through a robust database-backed system.

In this python project, we will build a grocery store management application. It will be 3 tier application,

**Front end:** UI is written in HTML/CSS/Javascript/Bootstrap
**Backend:** Python and Flask
**Database:** mysql



**Prerequisites**
Python 3.x
MySQL Server

**Download MySQL Installer:**

https://dev.mysql.com/downloads/installer/

**Install Required Package**
pip install mysql-connector-python
pip install flask

**Run the Application**
Configure MySQL database settings in sql_connection.py
Start the Flask server
python server.py
Open the UI in your browser.

**Installation**
Clone the repository:
git clone https://github.com/Bhavani0915/Grocery_Store_application_python_project.git
Navigate to the project directory:
cd Grocery_Store_application_python_project
Install dependencies:
pip install -r requirements.txt
Configure MySQL database credentials.
Run the application:
python server.py
